# Data Structures and Algorithms

This repository contains implementations and tests for various data structures and algorithms in Python. I've organized the code into different folders based on the category of the data structure or algorithm.

I created this repository for personal reference, and I hope it also serves as a helpful resource for others who want to understand DSA.

## Installation

To get started with this repository, follow these steps:

1. Switch to the root directory of the repository.
2. Run `pip install` to install dependencies.
3. Run `pip run test` to execute the test suite.

Feel free to explore the code and tests to understand different data structures and algorithms.

## Usage

You can explore each problem folder to find Python files containing solutions by category-wise. Each solution file is named according to the technique used.

Feel free to explore, use, and contribute to this repository to improve the implementations or add new problems.
